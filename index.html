Futur réseau social pour l'insertion professionnelle, RESONANCES est un projet de l'Université Rennes 2. Ouvert à tous les publics, il sera la plateforme pour la communication entre les publics de l'Université et les professionnels.

Ce projet est piloté par le SUIO-IP de l'Université Rennes 2, dans le cadre du projet "IDEFI - 2PLG".

Pour être informé sur le projet et faire partie des premiers participants, vous pouvez demander une invitation.
